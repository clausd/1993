Welcome back, 1993 - I've missed you!

1993 is a library trying to return the web to 1993 - way back when hypertext, the document, ruled the web.

We believe the web was better back then. The architectural scope of the web was bigger. But we like web apps too. We just want to build them the old fashioned way - by writing documents.

1993 wants you to think documents, think REST. Web apps should be user editable documents, loosely joined - and nothing else.
